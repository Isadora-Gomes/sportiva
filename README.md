# Sportiva

**Sportiva** é uma loja virtual dedicada ao universo esportivo, oferecendo uma grande variedade de produtos, sempre com foco em **praticidade, segurança e uma experiência fluida** para o usuário.

O aplicativo foi desenvolvido para ser intuitivo, moderno e funcional, permitindo que qualquer pessoa navegue e realize compras de maneira simples.

---

## Funcionalidades

-  **Visualizar produtos**  
  Explore diferentes categorias e encontre itens esportivos com facilidade.

-  **Comparar preços**  
  Analise valores entre produtos para decidir a melhor opção de compra.

-  **Realizar compras**  
  Processo de compra prático, rápido e com foco na melhor experiência possível.

-  **Adicionar, editar e excluir produtos**  
  Funcionalidade que possibilita gerenciar o catálogo, adicionando novos produtos, editando detalhes existentes e excluindo itens quando necessário.
---

## Tecnologias Utilizadas

- **TypeScript**
- **React Native** 
- **Supabase** (banco de dados)
- **CSS**
